BCParks.ca ETL Services


## Setup

```
    npm install
```

## Run all scripts

```
    npm run start
```

## Run BC Wildfire Service Bans

#### with data from https://openmaps.gov.bc.ca
```
    npm run start bcwfs
```

#### with test data stored in a file
```
    npm run start bcwfs ./data/bcwfs-response.json
```
